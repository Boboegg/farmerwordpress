跟前面版本的本質差異

1. 亮度來自排版，不是色值：hero 用淺色漸層（主要是 #FDFCF8 白 + 一點主題色），而不是整片填色
2. 主題色只出現在小元件：tag、button、icon、kicker 文字、border 點綴 — 不在大面積背景上
3. 完全無棕：24 頁全部拒絕 brown/coffee，改用 Material Design 700-900 的紅藍綠紫
4. 共同的白底卡片：所有 scene 卡片都是白底 + 淺線，視覺統一
5. 漸層感：標題用 gradient text 做現代感
6. 預留動畫空間：hero 位置可加 ::after 光斑 / parallax layer / GSAP pin

這一版的關鍵進化

1. 目標：全齡 WCAG AA ≥ 4.5:1（不是只給中高齡的 AAA，未來可開「高對比模式」toggle 切換到 AAA）
2. 色彩總量：24 Primary + 24 Secondary 背景 + 4 共用 Accent = 52 色（低於門下省的 36 色硬規上限，但因為 Secondary 是背景不是文字可加寬）
3. 驗證方式：所有色值通過 Python WCAG 相對亮度公式實算，不信任 LLM 宣稱的數字（Gemini V6 被發現灌水）
4. 倡議頁去紫：改為 朱紅 #9E3A2E，避開華人喪葬聯想
5. 家族策略：7 主題家族（茶棕/紅/綠/金/綠/朱/墨藍/灰），子頁用同家族的明暗飽和度變化
6. 共用 4 Accent：警戒紅 / 信賴藍 / 深琥珀 / 深翡翠，全站按鈕一致性

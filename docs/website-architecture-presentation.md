# 尊嚴農業 — 網站架構與內容說明

> **計畫全名**：農業尊嚴：共建雲嘉農業工作者職業健康社會安全網
> **執行團隊**：國立中正大學 × 臺大醫院雲林分院
> **實踐場域**：嘉義縣民雄鄉
> **網站定位**：農業健康與安全知識樞紐（Knowledge Hub）

---

## 一、為什麼需要這個網站？

### 台灣農業面臨的嚴峻挑戰

| 數據 | 說明 | 來源 |
|------|------|------|
| **67.3 歲** | 農保被保險人平均年齡 | 台灣農保統計 |
| **5×** | 農業致死率為全行業平均的 5 倍 | CDC/NIOSH, 2024 |
| **38.5 萬** | 全球每年急性農藥中毒人數 | ILO |
| **30%** | 農民職業災害保險投保率 | 勞保局統計 |
| **56%** | 農業致死者中 ≥ 55 歲占比 | CDC/NIOSH |
| **3%** | 新住民農民職災保險投保率 | 勞保局統計 |

### 傳統農業資訊網站的問題

- 政令宣導式條列網頁，缺乏分眾設計
- 資訊被動、單向，無互動工具
- 缺乏國際數據佐證與學術嚴謹度
- 農民在數位近用與健康識能上面臨多重障礙

## 二、解決方案：矩陣式資訊架構

捨棄傳統條列網頁，採用**矩陣式資訊架構（Matrix Information Architecture）**：

```
                    ┌──────────────────────────────────────────────┐
                    │           七大資料庫主選單（縱軸）              │
                    │  01關於 02職安 03健康 04保險 05培力 06倡議 07研究 │
    ┌───────────────┼──────────────────────────────────────────────┤
    │  新手務農 Start │   依使用者身分，動態撈取對應縱軸模組的精選內容    │
    │  青農創業 Pro   │                                              │
    │  資深農友 Senior│                                              │
    │  一般公眾 Public│                                              │
    └───────────────┴──────────────────────────────────────────────┘
    （橫軸・分眾入口）          （縱軸・結構化資料庫）
```

- **橫軸** = 分眾入口（依生命週期分流）
- **縱軸** = 結構化資料庫（依知識領域組織）
- **首頁** = 分流大廳（The Lobby），降低認知負荷

## 六、技術架構

### 技術堆疊

| 項目 | 技術 |
|------|------|
| CMS | WordPress + Astra Child Theme |
| 前端 | 純 HTML/CSS（無 build 工具） |
| CSS | Custom Properties（設計 Token）+ 前綴隔離 |
| 動態功能 | PHP Shortcodes（Code Snippets 插件） |
| 部署 | `scripts/deploy.sh` → WP-CLI |

### CSS 前綴隔離制度

每頁使用獨立 CSS 前綴避免跨頁衝突：

| 頁面 | 前綴 | 頁面 | 前綴 |
|------|------|------|------|
| 職業安全 | `prev-` | 溫室專區 | `gh-` |
| 健康促進 | `hp-` | 省工農機具 | `mach-` |
| 經濟與保險 | `money-` | 個人防護具 | `ppe-` |
| 倡議 | `adv-` | Podcast | `pod-` |
| 研究成果 | `res-` | 分眾入口 | `start-`/`young-`/`senior-`/`public-` |

### WordPress 關鍵規範

**必須**：所有 HTML 頁面用 `<!-- wp:html -->` 和 `<!-- /wp:html -->` 包裹，否則 CSS 會跑掉。

**禁止**：`body{}`、`*{margin:0}`、`.container{}`、`html{}` — 會覆蓋 Astra 主題樣式。

### 部署流程

```
scripts/deploy.sh
  ├── Stage 1: CSS → Astra Child Theme
  ├── Stage 2: Shortcodes → Code Snippets（跳過）
  └── Stage 3: HTML → WP-CLI（依 page-map.json）
```

## 九、社會影響力與政策連結

### 覆蓋範圍

- 嘉義縣民雄鄉為核心實踐場域

- **日本** — 高齡農業社會安全網經驗
- **韓國** — 農業職業健康政策
- **澳洲** — 農場安全標準

# 尊嚴農業 — 網站架構與內容說明

> **計畫全名**：農業尊嚴：共建雲嘉農業工作者職業健康社會安全網
> **執行團隊**：國立中正大學 × 臺大醫院雲林分院
> **實踐場域**：嘉義縣民雄鄉
> **網站定位**：農業健康與安全知識樞紐（Knowledge Hub）

---

## 一、為什麼需要這個網站？

### 台灣農業面臨的嚴峻挑戰

| 數據 | 說明 | 來源 |
|------|------|------|
| **67.3 歲** | 農保被保險人平均年齡 | 台灣農保統計 |
| **5×** | 農業致死率為全行業平均的 5 倍 | CDC/NIOSH, 2024 |
| **38.5 萬** | 全球每年急性農藥中毒人數 | ILO |
| **30%** | 農民職業災害保險投保率 | 勞保局統計 |
| **56%** | 農業致死者中 ≥ 55 歲占比 | CDC/NIOSH |

### 傳統農業資訊網站的問題

- 政令宣導式條列網頁，缺乏分眾設計
- 資訊被動、單向，無互動工具
- 缺乏國際數據佐證與學術嚴謹度

## 二、解決方案：矩陣式資訊架構

捨棄傳統條列網頁，採用**矩陣式資訊架構（Matrix Information Architecture）**：

```
                    ┌──────────────────────────────────────────────┐
                    │           七大資料庫主選單（縱軸）              │
                    │  01關於 02職安 03健康 04保險 05培力 06倡議 07研究 │
    ┌───────────────┼──────────────────────────────────────────────┤
    │  新手務農 Start │   依使用者身分，動態撈取對應縱軸模組的精選內容    │
    │  青農創業 Pro   │                                              │
    │  資深農友 Senior│                                              │
    │  一般公眾 Public│                                              │
    └───────────────┴──────────────────────────────────────────────┘
    （橫軸・分眾入口）          （縱軸・結構化資料庫）
```

- **橫軸** = 分眾入口（依生命週期分流）
- **縱軸** = 結構化資料庫（依知識領域組織）
- **首頁** = 分流大廳（The Lobby），降低認知負荷

## 四、縱軸：七大資料庫主選單

| 編號 | 模組名稱 | 介面類型 | 核心功能 |
|------|----------|----------|----------|
| 01 | 關於計畫 | 靜態資訊 + 多媒體 | 團隊卡片 + 互動地圖 |
| 02 | 職業安全 | 網格卡片 + 互動工具 | NIOSH 控制層級 + 紅綠燈風險評估 |
| 03 | 健康促進 | 地理資訊 + 影音 | 互動人體圖 + 職醫地圖 |
| 04 | 經濟與保險 | 圖解 + 試算工具 | 保險試算器 + 法規懶人包 |

### 02 職業安全 — 深度解析

**頁面結構**：
1. Hero 區塊（標題 + 數據呼叫盒）
2. Tabs 切換：省工農機 ｜ 個人防護具 PPE
3. Grid Cards 數位型錄（含 PDF 圖卡下載）
4. 風險評估工具區

**互動工具**：
- 紅綠燈快篩系統（高/中/低風險）
- NIOSH 五層安全控制層級動態圖
- 四大危害深度檢核：生物・物理・化學・人因

**子頁面**：溫室專區 ｜ 省工農機具 ｜ 個人防護具

### 03 健康促進

- 互動式人體圖（點擊腰/膝/肩 → 播放復健 GIF）
- 職醫地圖（Google Maps API → 一鍵導航）
- 即時氣象高溫警報
- 急救紅卡（可下載列印）

### 04 經濟與保險

- 互動試算工具（輸入年齡/年資/傷病 → 補助/理賠估算）
- 涵蓋：職災保險、農保、老農津貼、農業保險
- 法規 → 步驟懶人包（流程圖）

### 05 培力與實踐（農學堂）

- 影片網格（每支標示時長與難易度）
- 工作坊清單 + 內建報名表單

### 06 尊嚴農業倡議

- 垂直時間軸串接倡議行動歷程
- 焦點團體會議紀錄
- 政策連署 + 社群分享按鈕

### 07 研究成果

- APA 第 7 版格式排版
- 每筆文獻附 [PDF 下載, X.X MB]
- Spotify / Apple Podcast 嵌入
- 子頁面：下載專區 ｜ 圖文資訊 ｜ Podcast ｜ 相關資源 ｜ 研究出版

## 六、技術架構

### 技術堆疊
- **CMS**：WordPress + Astra 主題
- **前端**：純 HTML/CSS（無 build 工具）
- **CSS**：Custom Properties（設計 Token）+ 前綴隔離
- **動態功能**：PHP Shortcodes
- **部署**：HTML 直貼 WordPress 程式碼編輯器

### 程式碼結構
```
farmerwordpress/
├── docs/           ← 架構藍圖、設計規格
├── css/            ← global.css 全站設計 Token
├── pages/          ← 各頁面 HTML 區塊
│   ├── About/
│   ├── occupational-safety/（含子頁）
│   ├── beginner-farmers/
│   ├── young-farmers/
│   ├── Public/
│   └── shared/sidebar.html
├── shortcodes/     ← WordPress PHP 短代碼
├── scripts/        ← 部署與匯入腳本
└── imports/        ← 批次匯入暫存區
```

### PHP Shortcodes
| Shortcode | 功能 |
|-----------|------|
| `my_news_loop` | 新聞迴圈 |
| `knowledge_podcast_grid` | Podcast 網格 |
| `weather_widget` | 天氣小工具 |
| `fb_feed_widget` | Facebook 動態牆 |
| `home_highlight_slider` | 首頁亮點輪播 |
| `news_magazine_layout` | 雜誌式新聞版面 |

## 七、學術嚴謹度與數據治理

### 引用規範
- 所有文獻嚴格遵守 **APA 第 7 版**格式
- **系統性排除中國大陸來源**數據

# 分眾入口頁面：改版設計說明

> **修改日期**：2026-02-23  
> **影響範圍**：4 個分眾入口頁面（新手務農 / 青農 / 資深農友 / 一般民眾）  
> **修改目的**：原頁面為相同模板換色，內容空洞且缺乏差異化。全面重寫為資料導向、情境化、分眾客製的登陸頁，讓不同受眾一進來就知道「這個計畫是什麼」和「我現在該看什麼」

---

## 一、改版前問題診斷

| 問題 | 說明 |
|------|------|
| **模板複製** | 4 頁 HTML 結構幾乎完全一致，僅替換色碼和標題文字 |
| **缺乏計畫介紹** | 訪客無法得知「尊嚴農業」是什麼、誰在做、在哪裡做 |
| **沒有數據支撐** | 頁面沒有任何統計數字或研究佐證，說服力薄弱 |
| **導航無情境** | 連結列表沒有脈絡，訪客不知道為什麼要點進去 |
| **分眾無差異** | 新手和資深農友看到一模一樣的頁面結構和語氣 |

---

## 二、改版設計原則

### 2.1 共用六段結構

所有 4 頁共享以下骨架，但每段的**內容、數據、語氣、CTA**完全依受眾客製：

| # | 區段 | HTML 結構 | 功能說明 |
|---|------|-----------|----------|
| ① | **Hero** | `.[prefix]-hero` | 情境開場白（一句話直擊痛點）＋ 副標語 ＋ 1 段計畫簡介 |
| ② | **數據震撼區** | `.[prefix]-stats` | 3 張統計卡（數字 + 說明 + 來源標註）＋ 底部補充說明 |
| ③ | **場景導航** | `.[prefix]-scenarios` | 4 張情境卡，每張含：圖示 ＋ 標題 ＋ 敘事段落（含數據）＋ 行動連結 |
| ④ | **計畫簡介** | `.[prefix]-about` | 1 段文字介紹 ＋ 4 行動支柱圖示（基礎調查 / 制度研究 / 健康教育 / 社會倡議） |
| ⑤ | **快速行動** | `.[prefix]-cta` | 2 顆按鈕：主要 CTA ＋ 次要 CTA |
| ⑥ | **緊急 + 探索** | `.[prefix]-emergency` / `.[prefix]-explore` | 119 / 1925 / 113 ＋ 4 頻道探索卡 ＋ 固定底部 3 按鈕導覽 |

### 2.2 技術規格

| 項目 | 規格 |
|------|------|
| 最大寬度 | `820px`（全頁面統一） |
| 響應式斷點 | `640px`（Mobile-First） |
| 字型 | Noto Sans TC + Lato + Lora（Google Fonts） |
| 圖示 | Font Awesome 6.4.2 CDN |
| CSS 命名 | 前綴隔離，避免跨頁衝突 |
| 底部導覽 | `position: fixed`，3 按鈕，隨頁面可見 |

## 四、特殊設計說明

### 4.1 資深農友的無障礙加強

| 項目 | 規格 |
|------|------|
| 基礎字級 | `18px`（其他頁面為瀏覽器預設 16px） |
| CTA 按鈕高度 | `56–60px`（其他頁面 52px） |
| 底部導覽按鈕高度 | `60–64px`（其他頁面 56px） |
| 電話號碼字級 | `1.5rem`（其他頁面 1.3rem） |

### 4.2 一般民眾的引言卡

一般民眾頁面在數據區下方額外設置引言卡（`public-quote-card`），引用陳宗延醫師（臺大醫院雲林分院）的話語，作為情感觸發點。此元素僅此頁獨有。

### 4.3 緊急資源

## 六、WordPress 操作指引

### 部署方式

1. 進入 WordPress 後台 → 頁面 → 找到對應頁面
2. 切換至**程式碼編輯器**（不是可視化編輯器）
3. **全選**原有內容 → **貼入**對應檔案的完整 HTML
4. 發佈 / 更新

### 注意事項

- CSS 以前綴隔離（`start-` / `young-` / `senior-` / `public-`），不會互相衝突
- Font Awesome 和 Google Fonts 透過 CDN 載入，已在每頁頂部引入
- `position: fixed` 的底部導覽需確認不被佈景主題的 footer 遮蓋
- 頁面底部留有 `100px` padding 為底部導覽預留空間
